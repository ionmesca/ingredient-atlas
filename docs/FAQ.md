# FAQ

## Is this an API?

No. It is a dataset and a small resolver package. You can download files or use hosted Hugging Face URLs.

## Can I use this in a commercial app?

The public release direction is CC0-1.0 for metadata and images. Check the final dataset card before using it.

## Are the images AI-generated?

Yes. The dataset says this clearly. Images are reviewed on a best-effort basis.

## Is nutrition data exact?

No. Some nutrition is USDA-backed. Some is best-effort or missing. Do not use it as medical, allergy, or dietary advice.

## Why not use Food-101?

Food-101 is useful for dish classification. Ingredient Atlas is for ingredient-level app assets.

## Why not use Open Food Facts?

Open Food Facts is excellent for packaged products. Ingredient Atlas is for clean ingredient images and app metadata.

## How do I report a bad image?

Open an issue after launch or email hello@ionmesca.com.
