# Ingredient Atlas v0.1.0

Initial public release candidate.

## Included

- 1,673 ingredient records
- 5,019 image files
- WebP thumbnails and PNG fallbacks
- JSONL, Parquet, full manifest, compact manifest, and checksums
- small npm resolver package
- issue templates for corrections

## Notes

Images are AI-generated and reviewed on a best-effort basis.

Nutrition metadata is best-effort and should not be used as medical, allergy, or dietary advice.
